Detailed Description

signed real format

signed real format is used to store float data its three portions this structure very helpful to convert from float to real_t and vice versa before sending in a network bus or displaying on a LCD /GLCD display
for example a float number -23.58 decimal holds the decimal part 58 integer holds the integer part 23 sig holds the sign 0:positive,1: negative
