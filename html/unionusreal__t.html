Detailed Description

unsigned real format (small size 8bit)

unsigned real format is used to store float data its two portions this structure very helpful to convert from float to ureal_t and vice versa before sending in a network bus or displaying on a LCD /GLCD display
for example a float number 137.76 decimal holds the decimal part 76 integer holds the integer part 137
