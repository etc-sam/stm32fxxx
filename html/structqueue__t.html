a queue struct for a ring buffer

the ring queue buffer consist of a pointer to fixed array usually with a total capacity and a length defines shows how data are in the queue, while index shows the position of the read data

Parameters
bufa pointer to a memory block of bytes to be read/ written
capacityit defines the phsical total size or capacity of that block of memory
pWRa pointer to the write position "the tail" within the queue
pRDa pointer to the read position "the head" within the queue
pTRa uart pointer to the "head" in the queue , just to print on the uart
