Detailed Description

data structure Bit definition for I2C_OAR1 register

this structure allows access at bit or byte level for the I2C own address register 1

Parameters
val16-bit value that represent the half word I2C_CR1 register
ADD01-bit Bit 0 of 10 bit Interface Address
ADD1_77-bit Bit[1..7] 7-bit Interface Address/10 bit Interface Address
ADD8_92-bit Bit[8..9] 10-bit Interface Address
res5-bit reserved not used bits
ADDMODE1-bit Addressing Mode (Slave mode)
